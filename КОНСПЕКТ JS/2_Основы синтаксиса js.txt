Продолжаем погружаться в JavaScript 2020 и сегодня познакомимся со структурой и основами синтаксиса языка. Проще говоря с правилами написания JS кода. Программа на JavaScript – это последовательность инструкций (statements). Инструкция – это некая законченная запись, команда которую мы отдаем на выполнение. При написании инструкций, их принято разделять между собой точкой с запятой. При этом в JavaScript допускается не ставить точку с запятой в конце инструкции, если следующая инструкция будет начинаться с новой строки. Но это работает не во все случаях. Инструкции JavaScript могут состоять из: значений, операторов, выражений, ключевых слов и комментариев. Для большинства этих частей запланирован свой отдельный выпуск. Сегодня мы только слегка познакомимся с ними. Строгий режим — "use strict". Когда появился ECMAScript 5 (ES5). Стандарт о котором я говорил в прошлом уроке. В JS появились новые возможности и изменились некоторые из существующих. Чтобы устаревший код работал, как и раньше, по умолчанию подобные изменения не применяются. Поэтому нам нужно явно их активировать с помощью специальной записи: "use strict".
